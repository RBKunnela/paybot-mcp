# Design Squad Skill

Unified design squad with 31 specialist agents organized in 3 tiers.

## Quick Start

Invoke the Design Chief to route your request:

```
@design-chief "your design mission here"
```

Or call a specialist directly:

```
@design-chief route brand strategy for new product
@design-chief route design system audit
@design-chief route thumbnail optimization
```
