# Inference Gap Persuasion — Shared Knowledge Base

> **Skill Reference:** `/inference-gap-persuasion` — invoke for full framework
> **Full Research:** `docs/research/2026-03-25-inference-gap-persuasion/02-report.md`

## TL;DR

Present Piece A and Piece B WITHOUT connecting them. The audience's brain generates Connection C on its own. Because C was self-generated, it bypasses critical defenses and is irresistible.

## The Architecture

```
PRESENT (A) → GAP → PRESENT (B)
                ↓
        AUDIENCE GENERATES (C)
                ↓
        C BYPASSES CRITICAL DEFENSE
```

## 5 Mechanisms

1. **Generation Effect** — Self-generated info remembered 2-6x better
2. **Ownership Bias** — IKEA Effect for ideas
3. **Reduced Reactance** — Can't resist your own thoughts
4. **Confirmation Bias** — Audience was primed to reach C
5. **Closure Drive** (Zeigarnik) — Brain MUST close incomplete patterns

## Quick Application by Domain

**VIDEO:** Kuleshov thumbnails → Zeigarnik cold opens → asymmetric reveals → juxtaposition montage → close loops + open one for next video

**COPY:** Precise headline gap → confirm suspicion → seeds of curiosity every 2-3 paragraphs → evidence pairs (never connect) → chain of acceptances → reader concludes

**SALES:** Mirror/label (Voss) → Socratic questions (Daley) → micro-commitments (Cialdini) → present two truths → strategic pause → prospect self-closes

**TEACHING:** Case A → Case B → "What do you notice?" → student generates principle → confirm with theory name

**STORYTELLING:** Plant detail early → add unrelated detail → let simmer → audience connects = 10x emotional impact

## Dosing Guide

| Type | Gaps | Close Timing |
|------|------|-------------|
| YouTube 10min | 4-6 | Every 60-90s, all before end |
| TikTok 60s | 1-2 | At payoff |
| Sales page | 8-12 | Gradually, final at CTA |
| Email sequence | 1/email | Next email opens |
| Sales call 30min | 3-5 | Prospect closes verbally |
| Lesson/module | 2-3 | Via discussion/exercise |

## Ethical Boundary

Both A and B must be TRUE. Conclusion C must be VALID. False pieces = manipulation = backfire (Persuasion Knowledge Model, Friestad & Wright 1994).
