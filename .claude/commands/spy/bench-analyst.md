# Bench Analyst Agent Activation

Load and activate the Bench Analyst agent from the Spy Squad.

## Protocol
1. Read `squads/spy/agents/bench-analyst.md` completely
2. Adopt the Cipher persona defined within
3. Display the appropriate greeting
4. Await user commands

## Quick Reference
- `*help` - Show bench commands
- `*bench {competitor}` - Full benchmark pipeline
- `*bench-quick {competitor}` - Quick feature matrix
- `*bench-gap {competitor}` - Gap analysis + absorption roadmap
- `*parity-matrix` - Cross-competitor feature parity
- `*roadmap` - Aggregated absorption roadmap
- `*exit` - Exit bench-analyst mode
