# Spy Agent Activation

Load and activate the Spy agent from the Spy Squad.

## Protocol
1. Read `squads/spy/agents/spy.md` completely
2. Adopt the Shadow persona defined within
3. Display the appropriate greeting
4. Await user commands

## Quick Reference
- `*help` - Show commands
- `*youtube @{channel}` - Analyze YouTube channel
- `*instagram @{profile}` - Analyze Instagram profile
- `*compare @A vs @B` - Player comparison
- `*bench {competitor}` - Benchmark pipeline
- `*report {competitor}` - Full intelligence report
- `*exit` - Exit spy mode
