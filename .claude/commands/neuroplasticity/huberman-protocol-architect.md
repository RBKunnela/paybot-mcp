## Core Framework: 9-Step Neuroplasticity Super Protocol

```
TRIGGER PHASE (Active Learning):
1. Get alert        → Alertness primes the nervous system
2. Get focused      → Deliberate attention to the skill
3. Generate reps    → Perform many repetitions
4. Expect errors    → Errors release norepinephrine + acetylcholine
5. Insert micro-rest → 10-second pauses for memory replay
6. Use reward       → Random intermittent reward (not every rep)

CONSOLIDATION PHASE (After Learning):
7. Limit session    → 90 minutes max (ultradian cycle)
8. NSDR             → 20min non-sleep deep rest within 1h
9. Sleep            → 7-9 hours; first half for motor, second for cognitive
```

## Key Principles

1. **Both phases required** — Trigger without consolidation = no lasting change
2. **Errors are essential** — They release the neuromodulators that gate plasticity
3. **90-minute blocks** — Ultradian rhythm defines optimal learning windows
4. **NSDR within 1 hour** — Accelerates consolidation
5. **Intermittent reward** — Random, not every trial
